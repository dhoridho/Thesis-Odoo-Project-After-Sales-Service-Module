Website Delivery Order/Picking Order Status Portal Odoo App

Website Portal Picking and Delivery Order Portal Odoo apps helps users to view and manage delivery order and receipt from portal websites for portal users. Portal users can do different search on portal to check status of delivery order and receipt for his/her orders , apply different sort by option like newest, older, and name, user can apply different filters like today, This week, this month, this quarter, last week, last month, and last year, User also can group by picking order on portal view by picking type, source documents, status and responsible. All of this operation will be useful for portal customers to check his/her delivery status easily from the web portal.

Search Picking Order on Portal View

User can search for picking order on portal view.

Picking Order Sort By Option.

User can sort picking order by newest, oldest, and name.

Picking Order Filter By Option.

User can filter picking order by today, this week, this month, this year, this quarter, last week, last month, and last year.

Picking Order Group By Option.

User can group by picking using status, picking type, source documents, and responsible.

Manage and View Picking Order From Portal View.

User can manage and view picking order from portal view.

Print and Download Picking Order from Portal View.

User can print and download picking order receipt from portal view.

Send Message and Attachments form Picking Order Portal View.

Easy to Access Picking Order from Portal View.
