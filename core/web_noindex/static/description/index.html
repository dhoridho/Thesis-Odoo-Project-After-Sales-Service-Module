Website Robots NoIndex NoFollow

Prevent robots and search engine crawlers from crawling your Odoo domain

Module Features

Automatically add the necessary code to all frontend web pages
No code, just install the module

Description

Frequently, companies use Odoo without a frontend website or only for internal purposes. So, there is no reason to let search engines to index your domain.

To solve it and stop indexing you can take a task to a developer or ask your content manager to block each webpage manually by deactivating the "Is Indexed" option. But actually, it's complicated, you can install this module and blocking all your frontend web pages automatically.

The module adds the meta tag "robots" as recommended by Google.
