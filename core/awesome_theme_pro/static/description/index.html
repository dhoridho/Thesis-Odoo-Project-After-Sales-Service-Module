Is this theme One-click installable?

Yes! You can install this theme in just one-click. Theme installation is very simple.

Login into your website and just install the Theme General module and Go to setting --> awesome theme. you can change the setting for this theme.

Do I resell or distribute the theme?
No! You can not resell or distribute this theme. You can use this theme only for your website.
Do I request for extra customization in theme after purchase?
Yes! We would love to provide you the best services possible. You can provide all customization requests on email and our technical team will get back to.
Do I get free updates and upgrades?
Yes, we provide free upgrades to our theme for the same Odoo Version. You can always download the latest version of the theme, replace your old theme module with the latest theme module and update the Theme.
