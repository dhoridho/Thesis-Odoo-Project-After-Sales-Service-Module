Venue Booking

This Application allows create venues, amenities, and bookings for venues

Select Booking From Menu

Go to Settings under Configuration and configure values for Venues, Amenities, Additional Charge

Create Venue Type from Venue Type menu under configuration.

Create Amenities

Venue Creation

Select Venue from Venue Menu

Add all essential details like venue name, capacity, Venue charge and additional charge based on days and hours.

While adding amenities we can select type for amenities and fix price for amenities on daily and hourly basis

If the type of amenitiy is inclusive cost of amenity will not be calculated.

If the amenity type is additional, cost of amenity will calculated.

Booking Creation

Select the Booking Enquiries or Confirmed bookings menu under Booking Main menu

Enter the details for booking.

If the customer is already in our customer list, select the customer

If the booking is from a prospective customer, Add the customer details in the Customer info Tab

At the time of booking confirmation the customer will be added to our customer list automatically.

Select the venue.

Enter the Booking From and To dates

If the To date is leeser than from date warning will be generated.

We can select the booking type.

If the booking type is day venue and additional charge based on day will be displayed.

If the booking type is hourly venue and additional charge based on hours will be displayed.

Total venue and additional charge will be calculated based on the duration of booking and values based on booking type

If the amenities we selected is inclusive in the venue package an error will be displayed

The unit price for the amenity will be will be based on the booking type.

Select the amenities and save the booking.

Currently status of booking is Enquiry.

Select the Confirm booking button.

In the pop up, there will be 2 options.

Select the link to an existing customer option, if the prospective customer is linked to any of our existing customers.

In the customer tab, Select the existing customer.

Select the Create new customer option if we want to add the prospective customer to the customer list.

Once the booking is confirmed, our prospective customer will change to new customer.

Booking status will be changed and some new buttons will appear.

Select the print button to get the pdf report of booking order.

Select the booking confirmation button to send the mail regarding confirmation to the customer.

Select the create invoice button to generate invoice for the booking.

In the pop up generated there is an option to add Downpayment either based on percentage or based on amount.

Select the create invoice button

If we want to view the invoice, select the create and view the Invoice

The Downpayment amount generated will be updated in the payment received field and amount due will be updated based on that.

Invoice count button will be generated in the Booking Order Form

The entire invoice option will generate the whole invoice

Check Availability

Click on Check Availability button to check venue avialabilty

Search by Venue

If venue selected, return calendar view of booking of selected venue

If venue and date selected, show message available or not available

Search by Others

If only amenity selected without date, show calendar view of booking with available venue.

If amenity and date selected, show message venue available or not available.

Also you can search by other filters like location, budget, seating etc,.

Reporting

Analysis Report

Select the Booking analysis menu from the reporting menu.

Analysis can be made on different measures like Total amount, amount due, advance received, customer, venue, Booking Reference

XML and PDF Reports

Select the Venue Booking Report menu from the reporting menu.

In the Pop up window, add the required details.

Select the Print PDF Button, to get the PDF report

Select the Export button to get the report in XML format
