Review User Access Rights in a Single View !

Overview

This module allows you to see all users access rights in a single view. This is visible for users whose have activated developer mode. And you can categorize users with Tag.

Added new field tag in the user form to categorize users.
Filter by User Tag
Group By User Tag
Activate the Developer Mode to access the menus.
Here you can select the groups as columns. Your configurations will be saved and that will be default at next time.
Report view with the selected columns
