Cost Estimation Sheet

◉ This module helps to create a Cost estimation sheet of project.
◉ Cost estimation sheet comprises of details of Material estimation, Labour estimation, and Overhead estimation.
◉ You can also create a bill of estimation sheet after approval by project manager.

Cost Estimation Report

Project

◉ Project comprises of Documents, Tasks, Cost sheet, Notes and Timesheets.

Project Analysis

Project Note

◉ This module helps to create project notes for projects to do the changes in project according to requirements.

Project Issue

◉ This module allows you to manage Construction and Work order Issues in projects.
◉ It is for labours who work on-site at construction and work contracting and allow them to raise any kind of issues related to works they are working on.

Work Order

◉ This module allows you to manage work order of projects and timesheets of employee.
◉ Work order also manages sale orders, work order notes, project issues, equipments requests and vehicle requests.

Work Order Note

◉ This module helps to create work order notes for work order to do the changes in work according to requirements.

Equipment Request

◉ This module helps to manage equipment requests for any work order. It also generates an invoice for that equipment request.

Vehicle Request

◉ This module helps to manage vehicle requests for any work order. It also generates an invoice for that vehicle request and also the driver can manage the invoices.

Employee

◉ This module helps to manage employees like Labours, Project manager, Driver, etc...
◉ Employee comprises of the details of the contract, timesheet, payslips and vehicle requests for the driver.

Employee Contract

◉ For payment of employee, it requires an employment contract which is also managed by this module. The employment contract also comprises of the details of salary structures.

Employee Payslip

◉ This module helps to create salary rules for different work types and also create employee payslip.
