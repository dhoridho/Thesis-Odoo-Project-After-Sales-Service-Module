Product Price Checker

Do you want to check the product price with product details quickly? This module allows you to check the product price using barcode no. User can enter barcode number by touch keyboard and it will be auto-scan through barcode scanner hardware attached, After a successful scan, You can see product information like product image, product code, product barcode, product sales price, available stock, product specification, product category. It can be useful for the customer also as a customer can easily check product information from the touchscreen or normal screen. we have a virtual keyboard so the user can use in touchscreen. everything is given configurable so the user can easily enable/disable as per his requirement from configuration screen, cheers!
