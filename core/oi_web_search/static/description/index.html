Advanced Search

"Make the Search Process Simple and Extensive"

This Module Provides Comprehensive Search Feature in All Applications of Odoo ERP..

Description

The Advanced Search allows users to benefit from the rapid and deep search process. This module extends the functionality of the Search engine to make search operation comprehensive and ensure an advanced user experience. Advanced Search provides wide range of filters and multi-criteria searches which allows users to search more rapidly, easily and deeply.

Key Features

  • Optimizes search performance with various search parameters.

  • Provides multi-level and multi-dimensional searchability.

  • High flexibility and discretion in search operation.

  • Accelerates search process.

  • Ability to select wide range of search criteria.

  • Ability to filter search results by wide range of parameters.

  • Fully integrated with all Odoo applications.
