superbar的属性

  • 在搜索视图xml中，编写superbar标签,
  • 位置: [left, right]
  • 视图模式: [kanban,tree,pivot,graph]
  • 字段:像普通的字段定义

字段定义属性

  • parent_key:父子关系的键字段名。留空将自动设置.
  • 级别:初始化需要扩展树的多少级别。默认是2
  • 限制:多少条记录显示ztree。默认是16
  • order:要订购的字段名称
  • 域:为数据设置域， eg: [[('is_company', '=', 1)]]
  • --
  • 或者你可以在.py文件中设置动态域.
  • 使用参数:searchpanel_domain_xxx, xxx是一个字段名.
