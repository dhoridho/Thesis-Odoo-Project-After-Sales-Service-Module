Advance Custom Filter

The Advance Custom Filter module categorizes your records according to the conditions you
have created by domain. You can create custom filters and set appropriate values for them. The filter can
sort out the records that meet the domain used for filtering the records from all the data.

Settings configuration

Select the given option from the drop-down menu to give access to advance custom filter.

To activate Advance Custom Filter, go to Settings > User & Companies > User.

If you give access to the Users, then the users can only create filters and group by themselves.

If you give access to the manager, then the manager can create filters and group
by for all the users.

Why should I use this app?

The "Advance Custom Filter" module is an essential tool for efficient data management as it allows you to categorize records based on specific conditions within a given domain. By creating custom filters and defining corresponding values, this module enables you to precisely sort and extract records that meet the specified criteria. This functionality enhances data organization, making it easier to isolate and analyze relevant information, ultimately streamlining your workflow and improving overall data accuracy.
