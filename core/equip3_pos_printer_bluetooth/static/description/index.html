POS Restaurant Bluetooth, Built-in, USB and IP Printer

This module Allows you to print POS Bar/Restaurant receipts directly using Bluetooth, Built-in, USB or IP Printer on SUNMI/Android devices

This module Allows you to contact up to 5 Bluetooth, USB or IP kitchen printers with your SUNMI/Android devices

Work with all SUNMI/Android devices (V2 PRO,T2 MINI,T2 .....)

  • Prints POS kitchen Receipts on up to 5 printers
  • Reprint POS Receipt and kitchen Category Receipts
  • Print POS Sales Report
  • Support automatic printing POS Receipt and kitchen Category Receipts
  • Support of Bluetooth, IP Network and USB printers
